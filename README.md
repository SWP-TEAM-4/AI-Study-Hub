# AIStudyHub
AI Study Hub là hệ thống hỗ trợ quản lý và chia sẻ tài liệu học tập tích hợp AI Chatbot. Ứng dụng giúp sinh viên lưu trữ tài liệu trên cloud, tìm kiếm nhanh tài liệu theo môn học và tương tác với AI để hỏi đáp nội dung học tập.

Hệ thống được xây dựng nhằm giải quyết vấn đề tài liệu phân tán ở nhiều nền tảng khác nhau như Google Drive, Messenger, Facebook Group hoặc Email, giúp việc học tập trở nên tập trung và hiệu quả hơn.

# Yêu cầu chức năng
1. Authentication
- Đăng ký tài khoản
- Đăng nhập
- Đăng xuất
- Quên mật khẩu
- Cập nhật profile
2. Document Management
- Upload tài liệu
- Xem danh sách tài liệu
- Tải xuống tài liệu
- Xóa tài liệu
- Chỉnh sửa thông tin tài liệu
- Xem chi tiết tài liệu
- Tìm kiếm tài liệu
- Lọc tài liệu theo môn học
3. Cloud Storage
- Upload file lên cloud
- Xem trạng thái upload
- Preview file
4. AI Chatbot
- Chat với chatbot
- Hỏi đáp về tài liệu
- Nhận câu trả lời AI
- Xem lịch sử chat
